# Mini SCADA – Power Substation Monitoring System

## Description
This project simulates a mini SCADA system for monitoring electrical parameters
such as voltage, current, power, and frequency.

## Technologies Used
- Python
- Flask
- HTML
- CSS

## How to Run
1. Install Python
2. Install Flask:
   pip install flask
3. Run:
   python app.py
